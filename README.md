# Car Booking Analysis Dashboard
A Tableau dashboard designed to analyze car booking performance across states, trip types, car models, and customer ratings.
#
<a href="https://github.com/deysweta970-hub/Healthcare-Analytics-Dashboard/commit/d14ac3b90e3d9500e79b69052281bc3975fe454c">Car Booking</a>
# Project Overview
This dashboard provides a complete analysis of car bookings across different states, car models, and trip types.
It helps understand booking trends, customer rating patterns, and high-performing regions/cities.
# Key Metrics
Total Bookings: 334
Total Revenue: ₹1,100,700
Completed Bookings: 307
Cancelled Bookings: 27
Average Rating: 5.889
# Insights from Dashboard
Top 5 Cities by booking completion
Trip Type Analysis (Airport Transfer, Local, One-way, Round trip, Outstation)
Car Type Rating (Luxury, SUV/XL, Sedan, MUV, Traveller)
Car Model Performance – Honda Amaze, Swift, Glanza, Ertiga, Nexon, etc.
State-wise Booking Distribution using a map visual
Weekly Booking Trends (Week 1 to Week 10)
# Purpose of the Project
This dashboard helps in:
Understanding customer booking behavior
Identifying top-performing cities
Monitoring booking completion & cancellation patterns
Improving fleet allocation & planning
Tracking customer satisfaction with ratings
